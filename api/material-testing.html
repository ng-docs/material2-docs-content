API reference for Angular Material testing

import {} from '@angular/material/testing';

Services

TestGestureConfig extends GestureConfig

An extension of GestureConfig that exposes the underlying HammerManager instances. Tests can use these instances to emit fake gesture events.

Properties
NameDescription

events: ANGULAR_MATERIAL_SUPPORTED_HAMMER_GESTURES

List of new event names to add to the gesture support list

hammerInstances: Map<HTMLElement, HammerManager[]>

A map of Hammer instances to element. Used to emit events over instances for an element.

Methods
buildHammer

Create a mapping of Hammer instances to element so that events can be emitted during testing.

Parameters

element

HTMLElement

emitEventForElement

The Angular event plugin for Hammer creates a new HammerManager instance for each listener, so we need to apply our event on all instances to hit the correct listener.
