Angular Material badge API 参考文档

import {MatBadgeModule} from '@angular/material/badge';

指令

MatBadge

Directive to display a text badge.

用于显示文字徽章的指令。

选择器： [matBadge]
属性
名称 描述
@Input('matBadgeColor')

color: ThemePalette

The color of the badge. Can be primary, accent, or warn.

徽章的颜色。可以是 primary、accent 或 warn。
@Input('matBadge')

content: string | number | undefined | null

The content for the badge

徽章的内容
@Input('matBadgeDescription')

description: string

Message used to describe the decorated element via aria-describedby

用于描述被 aria-describedby 装饰的元素的信息
@Input( matBadgeDisabled)

disabled: boolean

Whether the component is disabled.

该组件是否已禁用。
@Input('matBadgeHidden')

hidden: boolean

Whether the badge is hidden.

徽章是否隐藏了。
@Input('matBadgeOverlap')

overlap: boolean

Whether the badge should overlap its contents or not

徽章是否应与其内容重叠
@Input('matBadgePosition')

position: MatBadgePosition

Position the badge should reside. Accepts any combination of 'above'|'below' and 'before'|'after'

徽章应该放在哪里。接受 'above'|'below' 和 'before'|'after' 的任意组合
@Input('matBadgeSize')

size: MatBadgeSize

Size of the badge. Can be 'small', 'medium', or 'large'.

徽章的大小。可以是 'small'、'medium' 或 'large'。
方法
getBadgeElement

Gets the element into which the badge's content is being rendered. Undefined if the element hasn't been created (e.g. if the badge doesn't have content).

获取要在其中渲染徽章内容的元素。如果该元素尚未创建（例如，如果徽章没有内容），则为 undefined。
Returns
返回值
HTMLElement | undefined
isAbove

Whether the badge is above the host or not

徽章是否在宿主元素上方
Returns
返回值
boolean
isAfter

Whether the badge is after the host or not

徽章是否在宿主元素下方
Returns
返回值
boolean

类型别名

MatBadgePosition

Allowed position options for matBadgePosition

matBadgePosition 可接受的位置选项

type MatBadgePosition = 'above after' | 'above before' | 'below before' | 'below after' | 'before' | 'after' | 'above' | 'below';

MatBadgeSize

Allowed size options for matBadgeSize

matBadgeSize 可接受的大小选项

type MatBadgeSize = 'small' | 'medium' | 'large';
